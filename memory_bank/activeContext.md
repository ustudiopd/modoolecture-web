# 현재 작업 상황 (Active Context)

## 1. 현재 집중하고 있는 작업  
- **작업명**: 질문 보드 방송용 기능 완성
- **목표**: 질문 모달을 방송용으로 사용할 수 있도록 폰트 크기 조정 및 전체 화면 보기 기능 구현 완료
- **담당자**: AI Assistant

## 2. 다음 예정 작업  
- 질문 보드 추가 기능 개선
- 에디터 콘텐츠 저장 및 불러오기 기능 구현
- 이미지 업로드 기능 구현 (현재는 URL 입력만 가능)

## 3. 주요 이슈 및 블로커  
- 현재 특별한 블로커 없음
- GPT 답변 전체 화면 보기 기능 테스트 필요

## 4. 최근 완료된 기능
- ✅ ReadOnlyEditor 컴포넌트 생성 (Novel.sh 에디터 읽기 전용)
- ✅ QuestionModal에서 ReactMarkdown을 ReadOnlyEditor로 교체
- ✅ 폰트 크기 조정 (제목 유지, 질문/답변 30% 축소)
- ✅ 폰트 크기 토글 기능 (T 아이콘)
- ✅ 질문 네비게이션 (좌우 화살표, 키보드 단축키)
- ✅ 답변 전체 화면 보기 (Gemini/ChatGPT)
- ✅ 슬래시 커맨드 메뉴 (`/` 입력 시 메뉴 표시)
- ✅ 텍스트 선택 시 포맷팅 서브 메뉴 (굵게, 기울임, 밑줄, 취소선, 코드, 링크)
- ✅ 이미지 리사이즈 기능
- ✅ YouTube 동영상 임베드
