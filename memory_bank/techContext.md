# 기술 스택 정보 (Tech Context)

## 1. 프레임워크 및 라이브러리  
- **언어 및 버전**: TypeScript, Node.js v20.x
- **핵심 프레임워크**: Next.js 16.0.10 (App Router), React 19.2.0
- **데이터베이스**: Supabase (PostgreSQL 기반)
- **상태 관리**: React Hooks (useState, useRef 등)
- **UI 라이브러리**: Tailwind CSS 3.4.0
- **주요 라이브러리**: 
  - Novel.sh 1.0.2 (리치 텍스트 에디터)
  - Tiptap 2.2.4 (에디터 코어)
  - @tiptap/starter-kit, @tiptap/extension-image, @tiptap/extension-link, @tiptap/extension-youtube, @tiptap/extension-underline
  - @tiptap/suggestion (슬래시 커맨드용)
  - lucide-react (아이콘)
  - tippy.js (툴팁/팝업)
