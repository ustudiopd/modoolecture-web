# 현재 작업 상황 (Active Context)

## 1. 현재 집중하고 있는 작업  
- **작업명**: Novel.sh 기반 블로그 에디터 완성
- **목표**: Notion 스타일의 리치 텍스트 에디터 구현 완료
- **담당자**: AI Assistant

## 2. 다음 예정 작업  
- 블로그 에디터를 실제 블로그 작성 페이지에 통합
- 이미지 업로드 기능 구현 (현재는 URL 입력만 가능)
- 에디터 콘텐츠 저장 및 불러오기 기능 구현
- 에디터 UI/UX 개선 (더 많은 포맷팅 옵션 추가 등)

## 3. 주요 이슈 및 블로커  
- 현재 특별한 블로커 없음
- 이미지 업로드는 URL 입력 방식만 구현됨 (파일 업로드 기능은 추후 구현 필요)

## 4. 최근 완료된 기능
- ✅ 슬래시 커맨드 메뉴 (`/` 입력 시 메뉴 표시)
- ✅ 텍스트 선택 시 포맷팅 서브 메뉴 (굵게, 기울임, 밑줄, 취소선, 코드, 링크)
- ✅ 이미지 리사이즈 기능
- ✅ YouTube 동영상 임베드
